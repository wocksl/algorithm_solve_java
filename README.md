# Algorithm

**재찬이 알고리즘 연습장**

컴퓨터를 옮겨서 파일을 받으려는 상황 
1. git login 
2. git pull

인텔리제이에서 추가한 후 잔디를 올리고 싶은 경우
1. git status (올릴 파일 확인)
2. git add .
3. git commit -m "~~~"
4. git push

수정된 파일은 항상 'git status'로 확인

**문제를 풀고나면 항상 package로 분리하여 push 할것**
